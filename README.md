 Overview
This project builds an intelligent decision system to optimize factory allocation and reduce shipping lead time for a distributor.

It uses Machine Learning + Optimization logic to recommend the best factory for each product.



 Features

-  Predict shipping lead time using ML models  
-  Factory simulation engine  
-  Optimization (Speed vs Distance)  
-  Interactive dashboard using Streamlit  
-  Interactive charts using Plotly  
-  Factory location visualization  
-  Top 3 factory recommendations  


  Problem Statement

Traditional factory assignment leads to:
- High shipping delays  
- Increased logistics cost  
- No decision-making system  

This project solves it using predictive analytics and optimization.
